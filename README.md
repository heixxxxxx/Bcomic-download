## 解压下载器使用：

### 双击exe。
### 网页端打开b漫，进入漫画详情页（https://manga.bilibili.com/detail/mc....）
### 输入cookie（这个操作比较复杂）
- 	1.打开控制台（f12）
- 	2.上方标签页 选择网络（Network）
- 	3.刷新页面
- 	4.可以看到很多条数据，选择比较偏下面的一条，单击
- 	5.展开内容向下翻，找到Cookie: 复制后面的内容（注意不要把最后的回车换行也复制上了）
- 	6.粘贴到exe框内
- 	7.回车
### 输入comicId(看网址，mc后面的数字)
### 回车
### 输入话数(下载整本直接回车即可)
- 
### ps：如果账号没有购买，或者cookie输错了，只能下载预览页。
